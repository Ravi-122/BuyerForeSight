## Frontend Engineer Assessment
This is a frontend engineer assessment project. You can use **React** to implement the assignment.
## Core Task
Build a **User Directory Dashboard** using the API:

[https://jsonplaceholder.typicode.com/users](https://jsonplaceholder.typicode.com/users)

## Requirements

### 1. Dashboard
- Display users in a table or grid
- Show the following fields:
  - Name
  - Email
  - Phone
  - Company

### 2. Features
- **Search:** Filter users by name or email (client-side)
- **Sorting:** Sort users by Name and Company (ascending/descending)

### 3. User Detail Page
- Clicking a user row should navigate to a detail view
- Display full user information
## Tech Stack
- React 18
- React Router v6
- CSS
